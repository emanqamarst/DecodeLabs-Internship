# Task 1 - Cat Cafe Website
A cozy cat cafe website where coffee lovers and cat lovers unite. Built with pure HTML and CSS.

## Features
- Fully responsive navbar with sticky positioning
- Hero section with call to action buttons
- Meet the cats section with cat cards
- Interactive menu with categories
- Visit us section with location and cafe rules
- Reservation form
- Multi-column footer

## Technologies Used
- HTML
- CSS
